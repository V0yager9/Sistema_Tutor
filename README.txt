### Descrizione del progetto:
- Per compilare il progetto é stato usato CMake
- La compilazione del progetto avviene tramite riga di comando

### Istruzioni per compilare ed eseguire il progetto:
- Aprire un terminale all'interno della cartella ProgettoFinale
- Creare una nuova cartella inserendo il seguente comando:
	mkdir build
- Entrare nella cartella "build" tramite il seguente comando:
	cd build
- Una volta entrati nella cartella "build" si procede alla compilazione tramite i seguenti comandi:
	cmake ..
	make
- A compilazione terminata si può procedere all'esecuzione del programma nel terminale:
	./ProgettoFinale
